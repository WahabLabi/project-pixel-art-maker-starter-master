## Installation

Clone or download this repository to your local machine and then to play the game open the [index.html](index.html) file in your browser.

## Development

This game was built with HTML, CSS, JavaScript and jQuery. I used jQuery to manipulate the DOM and you can see my code in the [designs.js](designs.js) file.

Skills developed:

* [x] jQuery DOM manipulation
